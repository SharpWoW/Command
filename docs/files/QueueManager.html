Announce the current status of LFG to group.

  • _: Not used

Trigger method to announce status. This is because the status of LFG is not available straight after LFG_UPDATE event is fired.

Tables

Command.QueueManager
Table containing all QueueManager methods. This is referenced "QM" in QueueManager.lua.

Fields:

  • QueuedByCommand: True if player has been queued by a command, false otherwise.
  • Current: The current dungeon for which the group is queued.
  • Running: True if announce is running, false otherwise.
  • Time: Total time since Announce was called.
  • LastMode: Last not-nil value returned by GetLFGMode.
Command.QueueManager.Types
Contains information about various dungeon types. Each entry has an Alias table and an Id field. The Alias table contains names that this dungeon may be referenced by. The Id field is either a number or a function that gives the index for that dungeon type.

Fields:

  • ClassicRandom: Random Classic Dungeon
  • TBCRandom: Random Burning Crusade Dungeon
  • TBCHeroic: Random Heroic Burning Crusade Dungeon
  • LKRandom: Random Wrath of the Lich king Dungeon
  • LKHeroic: Random Heroic Wrath of the Lich King Dungeon
  • CataclysmRandom: Random Cataclysm Dungeon
  • CataclysmHeroic: Random Heroic Cataclysm Dungeon
  • Zandalari: Random Rise of the Zandalari Dungeon
  • Horseman: The Headless Horseman Hallow's Eve dungeon
  • BestChoice: Let the server decide what dungeon is best for the player
