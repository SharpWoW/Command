CM:HandleMessage (msg, sender, channel, target, isBN)
Handle a chat message.

Parameters:

  • msg: The message to handle.
  • sender: Player object of the player who sent the message.
  • channel: Channel the message was sent from.
  • target: Player or channel index.
  • isBN: True if battle.net message, false or nil otherwise.
